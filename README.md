# Приложение todo задач на React + TS, с использованием сервера на Express + TS.

Учебное приложение todo списка задач.
Поделено на две части backend и frontend составляющие.

UI фреймворк - Material UI

### Backend - server/

Выполняет простые задачи, предоставляет API для CRUD приложения.

- **GET** /api/todos - Получить все задачи

- **GET** /api/todos/:id - Получить задачу по id

- **POST** /api/todos - Создать новую задачу

- **PUT** /api/todos/:id - Обновить задачу

- **DELETE** /api/todos/:id - Удалить задачу

### Frontend - client/

Предоставляет интерфейс для просмотра и управления todo задачами.
- Просмотр
- Добавление
- Удаление
- Редактирование
